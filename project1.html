Gas Tank

In this project, I created a gas tank design highlighting several key SolidWorks skills. The model began with basic 2D sketches, which were extruded and revolved to create the cylindrical body and rounded end caps. The circular cutouts and holes in the filling port were created using sketching, cut-extrude features and circular patterns, enhancing both functionality and aesthetic appeal.

Created by Scott Whitbread with help from Herick Santos.
