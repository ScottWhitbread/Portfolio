Basic Computer Mouse

In this project, I created a basic computer mouse using CATIA V5â€™s Generative Shape Design workbench, focusing on utilizing Revolve and Extrude features to achieve a comfortable and ergonomic shape. The project emphasized smooth, continuous surfaces to create a functional and visually appealing design. I created the main body of the mouse by sketching the profile to define its ergonomic shape then extruding it. Using the Revolve tool, I generated a smooth, curved surface around an axis, forming the top shell of the mouse. This approach allowed for precise control over the curvature, creating a comfortable grip and natural resting position for the hand.
