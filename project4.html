Pipe System

In this project I created a pipe system in CATIA V5, focusing on the creation of fluid flow systems and precise part geometries. The design process began with 2D sketching to define the complex curves of the pipe sections, ensuring smooth transitions and proper alignment. The Rib feature was utilized to create the curved pipe sections by sweeping circular profiles along predefined paths, resulting in a consistent cross-section and smooth flow paths.

Created by Scott Whitbread with help from Herick Santos.
